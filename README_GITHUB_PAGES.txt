NOTDIENST ASYL – GITHUB PAGES INSTALLATION

1. GitHub-Konto erstellen oder anmelden.
2. Neues Repository erstellen:
   Name: notdienst-asyl
   Sichtbarkeit: Public
3. ZIP entpacken.
4. Diese Dateien hochladen:
   - index.html
   - manifest.json
   - sw.js
5. Im Repository:
   Settings > Pages
6. Unter „Build and deployment“:
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
   Save
7. Einige Minuten warten.
8. GitHub zeigt den Link, normalerweise:
   https://DEIN-BENUTZERNAME.github.io/notdienst-asyl/

Installation auf dem iPad:
1. Link in Safari öffnen.
2. Teilen-Symbol drücken.
3. „Zum Home-Bildschirm“ wählen.
4. „Hinzufügen“.

Wichtig:
- Die App benötigt beim ersten Öffnen Internet.
- Danach kann sie durch den Service Worker teilweise offline funktionieren.
- Vor dem Schließen immer Excel erstellen, weil Einträge nur in der geöffneten Sitzung bleiben.
